In a state, an Action can be specified with the format (entry|exit|during) Action.

For example, on entering MOM, the Event (from the Interface "Output") "advVel" should be triggered to advise the velocity of AUV to 1. The generated code is:
state MOM {
	entry advVel ! 1
}

For a transition, for example, the LRE shall transition from OCM to OCM if the Event "reqVel" (from the Interface "Input") is triggered, the returned value of the Event shall be stored in variable "x". Then the event "advVel" (from the Interface "Output") shall be triggered, to pass the value of "x". The generated code is:
transition t11 {
	from OCM
	to OCM
	trigger
	reqVel ? x
	action advVel ! x
}
